## 什么是 Clash for Android

**Clash for Android** 是一个基于 Clash 内核开发的安卓代理客户端，为安卓用户提供强大的网络代理功能。它是 Clash 生态中的重要组成部分，专门为移动设备优化。

### 主要特性

Clash for Android 具有以下核心功能：

| 功能特性 | 说明 |
|---------|------|
| 规则分流 | 支持 Rule、Global、Direct 三种模式 |
| 节点管理 | 支持多个节点的添加、编辑和管理 |
| 订阅导入 | 一键导入订阅链接，自动更新节点 |
| 自定义配置 | 支持 YAML 配置文件自定义 |
| 协议支持 | 支持 Clash Meta 内核的多种协议 |
| 自动测速 | 支持节点自动测速和智能选择 |
| 日志查看 | 实时查看代理日志和连接信息 |

### 应用场景

Clash for Android 适用于以下场景：

- **安卓手机用户**：需要在手机上使用代理的用户
- **平板电脑用户**：在平板设备上进行网络代理
- **开发者**：需要进行网络调试和测试
- **网络爱好者**：对代理工具感兴趣的用户

## 安装前准备

在安装 Clash for Android 之前，请确保您的设备满足以下要求：

- **Android 版本**：Android 5.0 或更高版本
- **存储空间**：至少 50MB 的可用空间
- **权限**：允许应用访问设备网络设置

## 常见问题

### Q: Clash for Android 是免费的吗？
A: Clash for Android 本身是免费开源的，但您需要自己提供节点或订阅源。

### Q: 如何导入订阅链接？
A: 在应用中点击"配置"，选择"新建配置"，粘贴订阅链接即可。

### Q: 支持哪些协议？
A: Clash for Android 支持 Clash Meta 内核的所有协议，包括 Shadowsocks、Trojan、Reality 等。

### Q: 如何设置代理规则？
A: 在应用中选择不同的模式（Rule、Global、Direct）或编辑自定义规则文件。
